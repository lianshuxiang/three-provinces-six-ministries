- ✅ 历史记录管理

**数据结构**:
```typescript
interface CostEntry {
  timestamp: string;
  taskId: string;
  agent: string;
  model: string;
  tokensIn: number;
  tokensOut: number;
  tokensCached: number;
  cost: number;
  cacheHit: boolean;
}
```

**成本计算**:
```
cost = (tokensIn / 1000 * inputPrice) +
       (tokensOut / 1000 * outputPrice) -
       (tokensCached / 1000 * (inputPrice - cachedPrice))
```

### 🎯 成本优化建议 (12.3 KB)

**职责**: 分析成本数据，生成优化建议

**核心功能**:
- ✅ 5 个维度分析
- ✅ 优化建议生成
- ✅ 预估节省计算
- ✅ 优先级排序
- ✅ 行动项清单
- ✅ 优化报告生成

**分析维度**:

1. **模型使用优化**
   - 检查 GLM-5 使用占比
   - 建议简单任务使用 GLM-4.7
   - 减少昂贵模型调用次数

2. **缓存命中率优化**
   - 目标: >70% 缓存命中率
   - 优化提示词结构
   - 增加可复用部分
   - 预加载常用上下文

3. **任务效率优化**
   - 降低平均任务成本
   - 优化任务拆分策略
   - 合并相似任务

4. **工作流优化**
   - 减少迭代次数
   - 提高并行度
   - 优化任务调度

5. **预算优化**
   - 检查预算超支
   - 审查任务优先级
   - 延迟非紧急任务

**优化建议结构**:

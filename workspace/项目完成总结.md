## 🚀 开始使用

```bash
# 克隆仓库
git clone https://github.com/lianshuxiang/three-provinces-six-ministries.git

# 进入目录
cd three-provinces-six-ministries

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
nano .env
# 设置你的 GLM API Key
# 从 https://open.bigmodel.cn/ 获取

# 运行示例
npm run example

# 查看成本
npm run cost:status
```
